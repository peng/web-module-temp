# web-module-temp

> web module project template

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao web-module-temp my-project
```

### From git

```bash
sao peng/web-module-temp my-project
```
